## 🔴 PASSO 1: ABRIR GOOGLE APPS SCRIPT

**Local**: Google Sheets → Extensões → Apps Script

```
1. Abra seu Google Sheets "Gestão de Depósito"
2. No menu superior: Extensões
3. Clique em: Apps Script
4. Abrirá uma nova aba com o editor de código
```

**Você verá uma arquivo chamado `Code.gs` ou similar**

---

## 🟠 PASSO 2: COPIAR CÓDIGO PRINCIPAL

**Arquivo para copiar**: `README.md`

```
1. Abra o arquivo README.md do GitHub
2. Selecione TODO o conteúdo (Ctrl+A)
3. Copie (Ctrl+C)
4. Volta para o Apps Script
5. Selecione TODO o conteúdo atual (Ctrl+A)
6. Delete (Delete)
7. Cole o novo conteúdo (Ctrl+V)
```

**Resultado**: Arquivo cheio de código

---

## 🟡 PASSO 3: ADICIONAR ARQUIVO DE AUTENTICAÇÃO

**Arquivo para adicionar**: `autenticacao_usuarios.gs`

```
1. No Apps Script, clique no botão "+" (novo arquivo)
2. Selecione "Arquivo JavaScript"
3. Nomeie como: autenticacao_usuarios
4. Abra o arquivo autenticacao_usuarios.gs
5. Copie TODO o conteúdo (Ctrl+A, Ctrl+C)
6. Cola no novo arquivo no Apps Script (Ctrl+V)
```

**Resultado**: Dois arquivos no Apps Script

---

## 🟢 PASSO 4: SALVAR E VOLTAR AO SHEETS

**Na guia Apps Script**:
```
1. Pressione Ctrl+S (salvar)
2. Aguarde "Saved" aparecer
3. Feche a guia do Apps Script
4. Volta para o Google Sheets
```

**Na guia Sheets**:
```
1. Recarregue a página (Ctrl+F5)
2. Aguarde carregar (pode levar 30 segundos)
3. Um novo MENU deve aparecer no topo
```

---

## 🔵 PASSO 5: EXECUTAR INICIALIZAÇÃO

**No Google Sheets**:

```
1. Abra o novo MENU que apareceu
   (Ex: 📦 DEPÓSITO)

2. Vá até: Sistema > Iniciar Sistema

3. Clique em "Iniciar Sistema"

4. Aguarde a conclusão
   (mensagem de sucesso deve aparecer)
   
5. Recarregue novamente (F5)

6. Popup de LOGIN deve aparecer automaticamente
```

---

## ⚡ PRÓXIMO: CRIAR SUA PRIMEIRA CONTA

**Quando LOGIN aparecer**:

```
1. Clique em: "✅ Criar Conta" (em azul)

2. Preencha os dados:
   • Nome Completo: Seu nome aqui
   • Email: seu@email.com
   • Senha: Mínimo 6 caracteres
   • Confirmar Senha: Repita a mesma
   • Perfil: Escolha "👨‍💼 Gerencial" (você)

3. Clique: "✅ Criar Usuário"

4. Verá mensagem de sucesso

5. Volta para LOGIN automaticamente

6. Use suas credenciais:
   • Email: seu@email.com
   • Senha: Que você criou
   
7. Clique: "🚀 Entrar"

8. HOME ABRE AUTOMATICAMENTE! 🎉
```

## 🆘 SE ALGO DER ERRADO

### LOGIN não aparece?
```
1. Recarrege (Ctrl+F5)
2. Verifique se os 2 arquivos .gs foram copiados
3. Verifique Console para erros
4. Repita a instalação
```

### HOME aparece em branco?
```
1. Menu > Sistema > Iniciar Sistema
2. Recarregue (F5)
3. Faça logout e login novamente
```

### Erro na criação de usuário?
```
1. Verifique se email já não está cadastrado
2. Senha tem 6+ caracteres?
3. Tente outro email
```

### Tema não aplicou?
```
1. Menu > Sistema > Iniciar Sistema
2. Recarregue
3. Se não funcionar, execute novamente
```
