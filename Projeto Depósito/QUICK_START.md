# ⚡ QUICK START - 5 MINUTOS

## 🎯 Você tem 5 minutos? Faça isto:

### 1️⃣ Copie o Código (1 min)
- Abra: `README.md`
- Selecione TODO o conteúdo (Ctrl+A)
- Copie (Ctrl+C)

### 2️⃣ Cole no Apps Script (1 min)
- Abra sua planilha Google Sheets
- **Tools → Script Editor**
- Delete tudo existente
- Cola o código (Ctrl+V)
- **Ctrl+S** para salvar

### 3️⃣ Execute (1 min)
```
Menu > 📦 Sistema > 🚀 Iniciar Sistema
```

### 4️⃣ Configure (1 min)
```
Menu > 📦 Sistema > ⚙️ Configurar Depósito
```
Preencha:
- Nome: `MEU DEPOSITO`
- Drive: `https://drive.google.com/drive/folders/...`
- Clique: **SALVAR**

### 5️⃣ Abra Home (1 min)
```
Menu > 🏠 Home
```

**Pronto! Seu sistema está rodando! 🎉**
