Sistema de gestão para depósitos com Google Apps Script + Google Sheets.

## Funcionalidades
- Estoque
- Vendas
- Compras
- Comandas
- Delivery
- Financeiro
- Dashboard otimizado (sem WebApp)
- WhatsApp Delivery MVP
- Logs e Backup

## Tecnologias
- Google Apps Script
- Google Sheets
- Google Drive API
- HTMLService (somente diálogos internos)
